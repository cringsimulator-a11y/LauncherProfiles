Known Issues & Status

March 11, 2025

Below is a list of currently known issues in Sonar Client. We're actively working on fixes — check back for updates.

⚠️ Note: Some features may be temporarily unavailable. We appreciate your patience while we resolve these issues.

Current Issues

📦
Fabric Not Downloading Broken

Fabric loader fails to download during installation. As a workaround, manually place your Fabric version in .minecraft/versions until a fix is released.

☕
Java Download Problems Broken

Auto-installation of Java 21 may fail on certain systems. If you encounter this, please install Java 21 manually from the official site and point Sonar Client to your Java path in settings.

👥
Profiles Hub Not Working Broken

The Profiles Hub is currently non-functional. Creating, sharing, and loading community profiles is unavailable. Local profiles are unaffected.

🚀
Premium Launcher Not Working In Progress

The Premium Launcher experience is currently under active development. Some premium features may not function as expected. This is a known limitation and will be resolved in an upcoming update.

✅
Login Fixed

Microsoft Account login is now working correctly. OAuth and refresh tokens are fully operational. If you were previously affected by login failures, please try again — no action needed on your end.

Last updated March 11, 2025 · More fixes on the way. Thank you for your patience! 🎉
