March Update — Bug Fixes & Java 21 Support

March 5, 2025

This month's update brings important stability improvements and full Java 21 support for all Minecraft 1.21+ versions.

Bug Fixes

  • Fixed crash when launching Fabric with custom Java path
  • Fixed profile stash folders appearing in the profiles list
  • Fixed version picker username field showing for MSA accounts
  • Fixed news feed not refreshing after first load

Improvements

  • Microsoft tokens are now saved permanently using refresh tokens
  • RAM allocation now correctly reads from launcher_data.json per profile
  • Version detection is now based on folder name (more reliable)
