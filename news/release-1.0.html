Sonar Client 1.0 Released!

March 11, 2025

We're thrilled to announce the official launch of Sonar Client 1.0 — a fully custom Minecraft launcher built from the ground up with our community in mind.

What's New

  • 🎮 Microsoft Account login — full OAuth with permanent refresh tokens
  • 👥 Friends system — see who's online and what they're playing
  • 📦 Profile system — create and share mod packs with your friends
  • 🔍 Modrinth integration — browse and install mods directly in the launcher
  • ☕ Java 21 auto-install — no manual setup required
  • 💬 Built-in chat — talk to friends without leaving the launcher
💡 Tip: Use the installer at SonarInstaller.py for a one-click setup that handles Python, Java, and all dependencies automatically.

Getting Started

Download and run SonarInstaller.py. It will set up everything in a Sonar Client/ folder. Once installed, launch python.py to start.

Known Issues

  • Snapshots are hidden by default — enable them in the version picker settings
  • Fabric version detection requires at least one Fabric install in .minecraft/versions
