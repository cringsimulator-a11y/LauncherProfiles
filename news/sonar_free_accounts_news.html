Free Alt Accounts — Coming Soon

Announced March 11, 2025  ·  Expected: Late April 2025

Play Minecraft for free — no purchase required

Sonar Client's next major update will let any user log in and play using a free alt account generated directly inside the launcher. No Microsoft account, no payment, no hassle.

This has been the most requested feature since launch. Starting in the next update, every Sonar Client user will be able to create and use a free "alt" account — a valid Minecraft session that lets you join servers, play singleplayer, and use all launcher features without owning a copy of Minecraft.

What's included

🆓
Free alt accounts Free
Generate a working Minecraft session username instantly from inside the launcher. No account required.
🔄
Alt switcher
Save multiple alts and switch between them with one click. Perfect for testing mods or playing on different servers.
🎮
Full launcher support
Alts work with vanilla, Fabric, and all profile types — the same as a paid account inside Sonar Client.
🔒
Premium accounts unchanged
If you already own Minecraft and use Microsoft login, nothing changes. Alts are a separate optional feature.
Note: Free alt accounts work on servers that have online-mode=true. Premium (Microsoft) / Alt accounts are required for online-mode servers like Hypixel.

Frequently asked questions

What exactly is a free alt account? ▼
A free alt account is a Minecraft session with a chosen username that doesn't require a Microsoft account or a paid copy of Minecraft. Sonar Client generates a valid offline-mode session token locally, letting you launch and play the game freely.
Can I play on any server with a free alt? ▼
Yes, Free alts work on all servers.
Will I lose my paid account if I use an alt? ▼
No. Your Microsoft account login and saved credentials are completely separate. You can switch between your paid account and any alt at any time from the launcher's account switcher — they don't interfere with each other.
How many alts can I create? ▼
There's no set limit planned. You'll be able to save as many named alts as you want locally inside the launcher. We may introduce a soft limit in future if needed, but the initial release will be unlimited.
Is this safe / will I get banned? ▼
No, There is no risk using Alt Accounts. However, individual server owners can ban usernames — always read a server's rules before playing.
When exactly is version 1.1 releasing? ▼
Our target is late April 2025. Development is in progress right now. If the timeline shifts, we'll post an update here. There's no exact date yet — we'd rather release it right than rush it.
Do I need to reinstall Sonar Client for the update? ▼
No. Sonar Client has a built-in auto-updater. When version 1.1 is released, you'll see an update prompt inside the launcher and can install it in one click. Your profiles, settings, and data will be preserved.

More details will be shared as development progresses. Stay tuned! 🎉
